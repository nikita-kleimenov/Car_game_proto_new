Данный пакет включает в себя 3 скрипта

1. SoundHolder - менеджер звука. 
	Позволяет задавать набор списков из необходимых для проекта звуков. 
	Для каждого звука имеется возможность настраивать уровень громкости посредством перетягивания ползунка, расположенного справа от него.
	В оглавлении каждого списка указывается название, с помощью которого в дальнейшем происходит доступ к рандомножу звуку списка.
	Доступ осуществляется посредством вызова через синглтон функции PlayFromSoundPack(string packName, isLoop = false).
	Вышеупомянутый метод возвращает ссылку на автоматически созданный и запущеный объект с AudioSource, который при
		1. (isLoop == true) - Удаляется автоматически после проигрывания клипа.
		2. (isLoop == false) - Необходимо удалять вручную.
	В том случае, если существует несколько списков с одинаковыминазваниями, выбран будет первый из них.


2. DataHolder - абстрактный класс.
	От данного класса наследуется SoundHolder. 
	Необходим для возможности передачи SoundHolder классу ScriptableInitializator через инспектор.


3. ScriptableInitializator - инициализатор.
	Необходим для инициализации синглтона SoundHolder через DataHolder.
	Для корректной работы необходимо установить значение ниже стандартного в Edit > Project settings > Script Execution Order
